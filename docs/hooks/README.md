# Hooks Documentation

This section covers custom hooks used throughout Seabreeze.

## Overview

Seabreeze provides custom React hooks for common functionality, organized by responsibility:
- **Chat Orchestration**: Main hooks for chat functionality
- **Streaming**: Real-time AI response handling
- **Persistence**: Database operations
- **Utilities**: Error handling and UX
